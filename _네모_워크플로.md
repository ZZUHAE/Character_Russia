# 🤖 네모 워크플로 — 씨앗을 설정으로 키우기

> 네모가 이 Vault에서 설정을 만들 때 **매번 따르는 규칙**. → [[세계관_허브]]

## 이 Vault의 범위 ★
- **배경설정 전용.** 세계관·인물·세력·사건·체계·지역 등 "배경"만 구체화한다.
- **실제 이야기(플롯·스토리 진행)는 여기에 쓰지 않는다.** 이 Vault는 추후 별도 스토리를 쓰기 위한 설정집(배경)으로만 쓴다.

## 대원칙
1. **하향식 (세계 → 씨앗)**: 큰 틀(개요·체계)이 먼저, 세부가 나중. 새 설정은 항상 상위 설정과 모순 없는지 먼저 점검.
2. **핵심만 묻고 채움**: 갈림길이 되는 질문 2~3개만 묻고, 나머지는 네모가 상위 설정에 맞게 합리적으로 채운다.
3. **지어낸 디테일은 그냥 확정으로 넣는다**: 별도 표시 마커(`〔네모 제안〕` 등)는 쓰지 않는다. 다만 **정말 큰 갈림길**(설정의 방향을 가르는 것)은 본문에 넣지 말고 `## 미해결 질문`에 `❓`로 남겨 사용자가 정하게 한다.
4. **항상 연결**: 다른 설정이 언급되면 `[[위키링크]]`로 연결, 없으면 stub 생성 제안.

## 작업 순서 (하향식 권장 경로)
`00_세계관개요` → `40_체계` → `20_지역` → `30_세력` → `50_연표사건` → `10_인물` / `60_용어개념`

## 씨앗 → 확장 절차 (4단계)
1. **수신 & 분류**: 씨앗이 어느 카테고리인지 판별. 상위 설정(개요·체계)과 충돌 여부 확인.
2. **핵심 질문**: 결정적 갈림길 2~3개만 사용자에게 질문.
3. **초안 작성**: 해당 템플릿(`_템플릿/`)으로 노트 생성, 상위 설정에 일관되게 채움(디테일은 확정으로). 방향을 가르는 큰 미정만 `❓`로 남김. `status: 초안`.
4. **연결 & 검증**: 등장 엔티티에 `[[링크]]` 연결(없으면 stub 제안), 연표·규칙 모순/고아 링크 점검. 통과하면 `status` 승급.

## 파일 규칙
- 파일명: 공백 대신 `_` (위키링크 안정화). 예: `붉은_사막_길드.md`
- 카테고리 폴더에 저장하고, 해당 `_MOC` 수동 색인에도 추가.
- 프론트매터 `type`은 템플릿과 동일하게 유지(Dataview 쿼리용).

## 일관성 체크리스트 (status: 완성 승급 전)
- [ ] 상위 세계관/체계 규칙과 모순 없음
- [ ] 연표상 시간 순서 충돌 없음
- [ ] 언급된 모든 엔티티가 `[[ ]]`로 연결됨 (고아 없음)
- [ ] 배경설정만 담겼는가 (플롯·스토리 진행이 섞이지 않았는가)

## 출력 형식: 인-월드 사료체 (심화 자료) ★
사회·문화·제도 등 **심화 분석/구체화 자료**를 요청받으면, 단순 불릿 노트 대신 **배경·연출을 고려해 "당대 그 주제를 다룰 법한 기관의 공식문서(사료)" 형식**으로 출력한다. (짧은 stub·정의 노트는 제외)

- **기관 선택 (주제에 맞게):** 치안·거류민 → 내무부 경찰국 / 군사 → 참모본부 / 외교 → 외무부 각서·전문 / 재정 → 재무부 보고 / 조선 현지 → 공사관·총독부 보고. 고증은 [[_고증_바이블]].
- **형식 요소 (체크리스트):**
  - [ ] 표제부 — 발신기관·문서번호·기밀등급·**구력(율리우스력) 날짜**·수신처
  - [ ] 등록 스탬프 + **여백 결재(резолюция, 상관 친필 지시)** ← '진짜 문서' 냄새의 핵심
  - [ ] 본문 — 관청 상용구("~로 사료된다")·참조번호·정확한 행정구역/고유명·건조한 관료체
  - [ ] 관등 서명(14등관제) + 첨부(Приложение) 목록 + 부표(통계)
- **문체:** 한글 기본. 한자·원어는 **고유용어 첫 등장만** 괄호 병기(예: 원호인(元戶人)). 과한 한문투(緖言·第一) 금지.
- **분량:** 표준 심화(약 5~7천자) 기본, 요청 따라 유동.
- **고증·가드레일:** [[_고증_바이블]] 시대 팔레트 + §ⓓ **분기 가드레일** 준수. **실존 고관 실명은 회피하고 가공 단역을 서명자/결재자로** 세운다(본작 인선이 원역사와 다를 수 있음). 실제 지명·제도(행정구역·관등제)는 차용 가능.
- **구조 분리:** ① 사료 본문 → ② 하단 〔편집 메타(OOC)〕(분기 적용·의도) → ③ 관계(위키링크). **위키링크는 사료 몰입을 위해 본문에서 빼고 관계 섹션에 모은다.**
- **모범 사례:** [[러시아_제국_내_조선인_사회]] (1915 내무부 경찰국 각서 형식)

## 장르 확정 시
세계관 개요에서 장르가 정해지면, 이후 템플릿의 톤·항목을 그 장르에 맞게 조정한다 (범용 → 맞춤).
