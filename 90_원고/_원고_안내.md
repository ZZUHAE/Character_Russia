# 📜 원고 폴더 — 사용 규칙

> 여기가 **소설 본문**이 사는 곳. 설계는 `80_집필`, 산문은 여기 — 섞지 않는다.

## 규칙
- 파일명: `NNN화_제목.md` (001부터). 예: `001화_네바강의_얼음.md`
- 템플릿: `_템플릿/회차.md`. type은 `원고`, status 트랙은 **`구상 → 초고 → 퇴고 → 탈고`** (설정 노트의 씨앗~완성과 다른 트랙).
- 부(部) 구분: 회차가 쌓이면 `1부_제목/` 폴더로 유연하게 도입. 파일명 번호는 전체 통번호 유지(링크 안정).
- 퇴고는 파일을 **덮어쓰며** 한다(버전은 git이 든다). 큰 개정 전엔 커밋으로 경계를 남길 것.
- **회차가 퇴고 이상으로 확정되면**: [[연대기]]에 작중 사건 기록 → [[현재_정세]] 갱신 → [[작중_타임라인]]·스레드 커버리지 갱신 → git 커밋 ([[_집필_규칙]] §7).
