3. ✅ 지방 권력 → **현상 병존** (도시=개혁/농촌=황실/변경=군정) → [[제국_지방행정과_총독제]]

9. ✅ 제국 통합 노선 → **지역별 혼합(패치워크)** (조선=협력 쪽 변경) → [[다민족_제국과_소수민족]]
10. ✅ 신흥 부르주아 → **대자본=내각 견고 / 중소상공=부동층** (불황 시 황실로) → [[신흥_부르주아와_상공계층]]

## G. 1차대전 배경 인터뷰 ✅ 완료 (2026-06-25)
> 사건의 핵심 배경인 대전을 블록 인터뷰(A~F)로 구체화. **배경 = 1914~1915 현재까지의 구조·질감만. 전쟁 결말·1915 이후는 진행 레이어 몫(의도적 미결).**

- **D(키스톤). 전쟁 지휘권 → ★ 차르 친정(최고사령관 친임).** 황실 위신·통수권 레버↑, 단 전쟁 결과를 차르가 personally 떠안음. 실제 전쟁수행능력은 내각 의존 → "차르가 지휘, 마리아가 먹이는 군대". 스타프카=황실파 노장(라돈스키)+내각편 기술장교(멜니코프) 이원. → [[1차_세계대전]]
- **A. 개전·초기전과(1914) → 비대칭.** 북부 동프로이센(대독)=**실제사대로 탄넨베르크 대패**(단 후방 안정으로 정치 붕괴 없음) / 남부 갈리치아(대AH)=**대승+카르파티아 압박**(AH 전쟁초반 위기). "독일엔 밀리나 오스트리아는 압도". → [[1914_개전]]
- **B. 전선 지리·1915 → 안정된 교착선.** 중부=독일 개입(고를리체급) 후 **폴란드 돌출부 질서있게 포기 + 갈리치아~폴란드 동부 선 안정**(실제사보다 서쪽, 대후퇴 완화) / 남부 캅카스=**사르카므시 우위**. → [[1915_동부전선_러시아군]]
- **C. 적·동맹.** AH=**다민족 균열 씨앗까지 강조한 노트 신설**(본작 다민족 러시아의 거울상, 1915 이후 균열 전개는 진행 몫) / 타전선=**실제사 가볍게 준용**(서부 교착·이탈리아 5월·불가리아 10월·세르비아). → [[오스트리아-헝가리_제국]]
- **E. 국내정치 효과 → ★ 균형 진자.** 장기전이 B를 미는 방향=**전황 따라 진자**(차르 친정 양날: 승전=황실 위신↑, 패색=차르 책임→내각↑, 단 패색 스윙은 차르도 위신 타격이라 완충) / 전시 권한=**경제·산업 동원 대폭, 정치권한(책임내각제)은 교착**. → [[교착_균형_모델]] · [[전시경제_총력동원]]
- **F. 해전.** 발트(대독)=**러일승리로 쓰시마 전멸 면해 보존됐으나 수세**(독일 공해함대 우위, 페테르부르크 방어 집중) / 흑해(대오스만)=**우위·봉쇄**, 해협은 미결 유지. → [[제국_해군_재건]]

> 의도적 미결(진행 레이어): 전쟁의 **결말·승패**, AH 균열의 실제 전개, 차르 친정 진자가 회랑을 벗어나는 시점, 흑해 해협 향방(14번과 연동).

## H. 소설 전환 인터뷰 ✅ 완료 (2026-07-11)
> Vault 주제 전환(설정제작기→대체역사 군상극 웹소설) 후 플롯·대한 라인 보강 인터뷰. 답안은 [[플롯_아웃라인]]·[[한성_정계의_역학]]에 반영.

**H-1. 플롯 (3문항)**
- **아크 귀결 → ★ 입헌군주제 전환 성공** (기본 항로. 값을 치르는 성공 — 완승 사이다 X, 교착의 무게 끝까지). → [[플롯_아웃라인]]
- **전쟁 궤적 → 원역사 유사 소모전, 혁명 없이 참전 지속.** 1905를 흡수한 개혁이 압력을 '입헌 요구'로 수렴. 입헌은 쿠데타가 아니라 **청구서**로 온다.
- **대한 라인 → 끝까지 회색 줄타기** (해답 없이 선택의 무게로 닫음) + 이쪽 배경 디테일 강화 결정 → H-2로.

**H-2. 대한 라인 보강 (3문항)**
- **황제 인물화 → ★ 고종 실명 등장** (러시아 측 니콜라이 2세 실명과 대칭. 가드레일: 1907 퇴위·1919 사망 등 원역사 운명 배제, 본작 행보는 창작). → [[고종]] · [[세계관_범위와_경계]] 빈칸 해동
- **황실 스탠스 → 줄타기 시도하나 무능·주도권 없음.** 그 진공이 역량 있는 인물들을 끌어올리는 장치. "옥새는 아무것도 결정하지 못하지만 무엇이든 뒤집을 수 있다." → [[한성_정계의_역학]]
- **인물 폭 → 앙상블 보강 5~6명.** 신설 6: [[고종]](비어 있는 정점) · [[서경환]](탁지부, 발전vs주권의 화신) · [[홍순익]](궁내부, 조정 보존) · [[차도형]](자주 급진의 얼굴) · [[윤정식]](행간의 주필) · [[백금옥]](사랑방의 객주).

> 의도적 미결(진행/집필 몫): 3부 '성공의 값' 구체, 승전 시점·방식, 급진의 첫 행동 표적, 밀지 정치의 첫 폭발 지점.

**H-3. 집필 직전 딥 인터뷰 ✅ (2026-07-11, 모호성 36%→14% 통과)** — 스펙: `.omc/specs/deep-interview-novel-preflight.md`
- **제목 확정 → 「북극성 아래」** (세계관명 승격).
- **최종 상태 → 완전한 입헌 전환, 영국식** (차르 존속·"군림하되 통치하지 않는다"·실권 완전 이양). 절반의 옥좌=2부 말 디딤돌로 강등. **변질 뒷맛 코드 기각** — 내각 시점 웹소설로서 승리는 진짜 승리.
- **성공의 값 → 과정의 비용**: 암흑 구간(2부 밑바닥) + 인적 값 1명(내각 기둥 중, 2부 축적으로 — 후보: 솔로비요프/멜니코프/박/라리오노바).
- **대한 결말 → 회색 줄타기 + 열린 문**(희망 코드, 깔끔한 해방 아님).
- **1부 클라이맥스 → 밀지 파문 연쇄** (고종 밀지 유출/위조 → 볼린스키 부상 → 베르호프 소환 위기).
- **플롯 3층 구조 신설**: 큰그림 → 부 블록(`80_집필/부_플롯/`, 블록=3~5화) → 회차. 각 층 네모 초안→승인. 기록·커밋은 "확정" 한마디에만.
- **독자 지향 → 균형형** (문체 바이블 현행).
> 의도적 미결: 인적 값의 '누구', 승전 세부(1918 작업 가정), 밀지 유출 경로(블록 3 집필 중 확정).
