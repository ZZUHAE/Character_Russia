## A. 러시아 내정·제도 ✅ 완료
1. ✅ 비밀경찰(오흐라나) 통제권 → **궁정·차르 직속** (황실의 정보 무기) → [[제국_보안부_오흐라나]]
2. ✅ 관료 인사권 → **내각 실질 장악** (차르는 거부권만, 오흐라나와 맞교환 구도) → [[제국_관료제와_부처구조]]
3. ✅ 지방 권력 → **현상 병존** (도시=개혁/농촌=황실/변경=군정) → [[제국_지방행정과_총독제]]
4. ✅ 검열권 → **내각·황실 별도 언론창구** (이원 매체, 상대 창구를 검열로 압박) → [[제국_검열과_언론통제]]

## B. 사회 구조 ✅ 완료

7. ✅ 토지개혁 → **자영농 육성 추진** (미르 해체 — 개혁이 농촌=황실 텃밭에 쐐기) → [[농촌사회와_농민층]]
8. ✅ 노동 → **포섭형 노동입법** (안전판이나 노동에 합법 조직력 부여) → [[노동자계급과_노동운동]]
9. ✅ 제국 통합 노선 → **지역별 혼합(패치워크)** (조선=협력 쪽 변경) → [[다민족_제국과_소수민족]]
10. ✅ 신흥 부르주아 → **대자본=내각 견고 / 중소상공=부동층** (불황 시 황실로) → [[신흥_부르주아와_상공계층]]

## G. 1차대전 배경 인터뷰 ✅ 완료 (2026-06-25)
> 사건의 핵심 배경인 대전을 블록 인터뷰(A~F)로 구체화. **배경 = 1914~1915 현재까지의 구조·질감만. 전쟁 결말·1915 이후는 진행 레이어 몫(의도적 미결).**

- **D(키스톤). 전쟁 지휘권 → ★ 차르 친정(최고사령관 친임).** 황실 위신·통수권 레버↑, 단 전쟁 결과를 차르가 personally 떠안음. 실제 전쟁수행능력은 내각 의존 → "차르가 지휘, 마리아가 먹이는 군대". 스타프카=황실파 노장(라돈스키)+내각편 기술장교(멜니코프) 이원. → [[1차_세계대전]]
- **A. 개전·초기전과(1914) → 비대칭.** 북부 동프로이센(대독)=**실제사대로 탄넨베르크 대패**(단 후방 안정으로 정치 붕괴 없음) / 남부 갈리치아(대AH)=**대승+카르파티아 압박**(AH 전쟁초반 위기). "독일엔 밀리나 오스트리아는 압도". → [[1914_개전]]
- **B. 전선 지리·1915 → 안정된 교착선.** 중부=독일 개입(고를리체급) 후 **폴란드 돌출부 질서있게 포기 + 갈리치아~폴란드 동부 선 안정**(실제사보다 서쪽, 대후퇴 완화) / 남부 캅카스=**사르카므시 우위**. → [[1915_동부전선_러시아군]]
- **C. 적·동맹.** AH=**다민족 균열 씨앗까지 강조한 노트 신설**(본작 다민족 러시아의 거울상, 1915 이후 균열 전개는 진행 몫) / 타전선=**실제사 가볍게 준용**(서부 교착·이탈리아 5월·불가리아 10월·세르비아). → [[오스트리아-헝가리_제국]]
- **E. 국내정치 효과 → ★ 균형 진자.** 장기전이 B를 미는 방향=**전황 따라 진자**(차르 친정 양날: 승전=황실 위신↑, 패색=차르 책임→내각↑, 단 패색 스윙은 차르도 위신 타격이라 완충) / 전시 권한=**경제·산업 동원 대폭, 정치권한(책임내각제)은 교착**. → [[교착_균형_모델]] · [[전시경제_총력동원]]
- **F. 해전.** 발트(대독)=**러일승리로 쓰시마 전멸 면해 보존됐으나 수세**(독일 공해함대 우위, 페테르부르크 방어 집중) / 흑해(대오스만)=**우위·봉쇄**, 해협은 미결 유지. → [[제국_해군_재건]]

> 의도적 미결(진행 레이어): 전쟁의 **결말·승패**, AH 균열의 실제 전개, 차르 친정 진자가 회랑을 벗어나는 시점, 흑해 해협 향방(14번과 연동).
