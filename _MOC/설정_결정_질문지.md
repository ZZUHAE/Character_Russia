## A. 러시아 내정·제도 ✅ 완료
1. ✅ 비밀경찰(오흐라나) 통제권 → **궁정·차르 직속** (황실의 정보 무기) → [[제국_보안부_오흐라나]]
2. ✅ 관료 인사권 → **내각 실질 장악** (차르는 거부권만, 오흐라나와 맞교환 구도) → [[제국_관료제와_부처구조]]
3. ✅ 지방 권력 → **현상 병존** (도시=개혁/농촌=황실/변경=군정) → [[제국_지방행정과_총독제]]
4. ✅ 검열권 → **내각·황실 별도 언론창구** (이원 매체, 상대 창구를 검열로 압박) → [[제국_검열과_언론통제]]
5. ✅ 두마 → **부분 전진** (입법권 일부 확대, 책임내각제는 못 감) → [[제국_두마]] · [[드미트리_쿠즈민]]

## B. 사회 구조 ✅ 완료
6. ✅ 신분제(소슬로비예) → **교착으로 못 건드림** (법적 신분 위에 능력·여론이 겹쳐 흐름) → [[제국_신분질서와_신민여론]]
7. ✅ 토지개혁 → **자영농 육성 추진** (미르 해체 — 개혁이 농촌=황실 텃밭에 쐐기) → [[농촌사회와_농민층]]
8. ✅ 노동 → **포섭형 노동입법** (안전판이나 노동에 합법 조직력 부여) → [[노동자계급과_노동운동]]
9. ✅ 제국 통합 노선 → **지역별 혼합(패치워크)** (조선=협력 쪽 변경) → [[다민족_제국과_소수민족]]
10. ✅ 신흥 부르주아 → **대자본=내각 견고 / 중소상공=부동층** (불황 시 황실로) → [[신흥_부르주아와_상공계층]]

## D. 외교·국제 ✅ 완료

## E. 동아시아·조선 ✅ 완료 (조선=구체화 축)
16. ✅ 대한제국 경제 → **러시아 이권 구체화(철도·부동항·삼림·광산·재정) → 지역별 회색 공존** (인프라=협력/자원=수탈) → [[보호국_대한제국의_사회와_경제]]
17. ✅ 판정권=회색지대 + **대한제국 공사=내각 텃밭(승전 전리품 논리), 협력파 [[니콜라이_베르호프]] 부임** / +조약 명칭 한러→**조러**로 변경 → [[조러_보호조약]] · [[대한제국_주재_러시아_공사관과_고문단]]
18. ✅ 일본 재무장 → **육·해군 분열** (단일 국책 없음, 통제 안 된 여러 불씨) → [[일본_육군_해군_파벌]]
19. ✅ 자주파 → **온건·급진 영구 분열** (급진=일본 위험한 연계, 최우성은 분열한 세력의 영수) → [[대한제국_자주파_노선_분화]]
