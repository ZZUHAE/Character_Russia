## 📊 진행 현황 — 유형별
```dataview
TABLE length(rows) AS "노트 수",
      length(filter(rows, (r) => r.status = "완성")) AS "완성",
      length(filter(rows, (r) => r.status = "구체화")) AS "구체화"
FROM "00_세계관개요" OR "10_인물" OR "20_지역" OR "30_세력" OR "40_체계" OR "50_연표사건" OR "60_용어개념"
WHERE type != "MOC"
GROUP BY type
```

## 🕐 최근 작업 (최신 15)
```dataview
TABLE status AS "상태", type AS "유형", dateformat(file.mtime, "MM-dd HH:mm") AS "수정"
FROM "00_세계관개요" OR "10_인물" OR "20_지역" OR "30_세력" OR "40_체계" OR "50_연표사건" OR "60_용어개념"
WHERE type != "MOC"
SORT file.mtime DESC
LIMIT 15
```

## 🧩 미완성 작업 큐 (씨앗·초안)
```dataview
TABLE status AS "상태", type AS "유형", description AS "요약"
FROM "00_세계관개요" OR "10_인물" OR "20_지역" OR "30_세력" OR "40_체계" OR "50_연표사건" OR "60_용어개념"
WHERE type != "MOC" AND (status = "씨앗" OR status = "초안")
SORT status ASC
```
> 비어 있으면 = 모든 노트가 구체화 이상. 건강한 상태.

## 🔍 무결성 점검 (세 표 모두 비어 있어야 정상)
**description 누락**
```dataview
TABLE type, status
FROM "00_세계관개요" OR "10_인물" OR "20_지역" OR "30_세력" OR "40_체계" OR "50_연표사건" OR "60_용어개념"
WHERE type != "MOC" AND (!description OR description = "")
```
**moc 누락**
```dataview
TABLE type, status
FROM "00_세계관개요" OR "10_인물" OR "20_지역" OR "30_세력" OR "40_체계" OR "50_연표사건" OR "60_용어개념"
WHERE type != "MOC" AND !moc
```
**modified 누락**
```dataview
TABLE type
FROM "00_세계관개요" OR "10_인물" OR "20_지역" OR "30_세력" OR "40_체계" OR "50_연표사건" OR "60_용어개념"
WHERE type != "MOC" AND !modified
```
> 폴더↔moc 교차 정합(폴더는 맞는데 moc를 엉뚱하게 지정)은 `_scripts/frontmatter_lint.py` 가 검사한다.
