# 📒 집필 확정 사실 원장

> **소설이 창조·확정한 세계 사실의 등록부.** canon(00~60, 동결)도 아니고 진행 상태([[현재_정세]])도 아닌 것 — 원고가 *만들어낸* 조문·문구·수치·소품이 여기 산다. 30화에서 "그때 협약 몇 조였더라"로 원고를 다시 뒤지지 않게, 어기면 바로 모순 나는 것들을 모은다.
> **원칙: 원고=진실의 원천, 이 원장=색인.** 어긋나면 원고 승. append 위주(수정 시 사유 한 줄). 갱신 = 확정 파이프라인 ④.5([[_집필_규칙]] §7).

## 📜 조약 · 법령 · 문서
| 항목 | 확정 내용 | 근거 |
|---|---|---|
| **함경 지선 협약** | 제7조: 러 수비대는 **역사(驛舍)·교량 경비에 한정**, **4년 자동 종료**. 담보 = **원산 해관세에 한정**. 제11조: **철도학당 견습 20명**(조선 측 관철). | [[003화_날짜가_두_개인_나라\|3화]] |
| **동원 입법 보완안·군수 공채 재가** | 재가 완료(솔로비요프 프레임 승리). **차르 친필 가필 1행 = "동원·운용은 통수 대권과 어긋남이 없어야"** — 모든 동원 권한에 회수 가능한 실(‘전시 대권’의 법문 교두보). | 5~6화 |
| **특별 군수 공채** | 두마 예산위 통과. 쿠즈민이 '전시 대권' 위헌 문장을 속기록에 각인. | [[007화_남는_말\|7화]] |
| **고종 밀지** | 낮 어보/밤 밀지 양다리. 1통 소각·1통 유출, 자주파에 '짐이 안다' 발송(**제2의 손** 경유, 홍순익도 모르는 직속 라인). | [[009화_두_개의_붓\|9화]] |
| **통문(通文)** | 목판 인쇄(손으로 벤 것이 아닌 반듯한 글씨·고른 먹), 도성 시전 처마 밑에 동시다발 게시. **논지 = 충군의 방향 전환**: "상감의 성심은 본디 아라사를 멀리하고 종묘사직을 지키려 하시거늘, 곁의 간신과 바다 건너 오랑캐가 어보를 억지로 끌어다 나라의 산과 물과 철길을 남에게 넘기니… 함경의 철도가 그 증거요, 상감께서 밤마다 침전에 등을 밝히시는 것은 그 원통함을 적으심이라." **비용 = 일본 공작망 자금**(판각·종이·먹·발품. 백금옥이 원가로 역산). | [[011화_도는_종이\|11화]]·13화 |
| **'베낀 밀지'** | 밀지의 **필사본의 필사본**(글자가 몇 군데 뭉개짐). **원본 소재 불명 = 반증 불가**가 그 성격("본 사람이 없으니 밀지지 / 바로 그게 무섭다는 게요"). ⚠️ **다나카의 본격 위조물은 블록 4 시점에 아직 미투입** — 유통된 것은 유출된 진짜의 필사 계열. | 11·13화 |
| **윤정식의 질문 사설** | 밀지·상감 두 글자 없이 도성 전체가 알아듣는 글. 본문: *"요사이 도성에 종이 한 장이 돈다. 어떤 이는 참이라 하고 어떤 이는 거짓이라 한다. 그런데 이상한 것은, 참이라 하는 이도 거짓이라 하는 이도 그것을 소리 내어 묻지는 못한다는 것이다. 물음이 죄가 되는 나라에서는 참과 거짓이 함께 늙는다. 우리는 다만 묻는다. 묻는 것도 죄인가."* → **"묻는 것도 죄인가" 여섯 글자가 철시의 명분·도회의 구호가 됨.** | 11·12·14화 |
| **효유문(曉諭文)** | 궁내부(홍순익)가 단독 기초. 내용은 "분명하지도 흐리지도 않은, 세 독자가 저마다 조금씩 실망할 문장". **어전 재가 없음("하루만 더 두고 보자") → 끝내 무산·미공표.** 세 독자 = 협력파(김영선)·저잣거리·정동 러 공사관. | 12·13·14화 |
| **'헛간 물건 봉인' 약조** | 최우성이 도회에 이름을 빌려주는 조건으로 차도형에게 받은 구두 약조 — **"그 헛간의 물건(원산 무기 2궤)은 그날 나오지 않는다."** 근거 논리: 쇠붙이가 끼는 순간 도회가 난이 되고, 난은 관에 사람 벨 명분을 준다. **14화에 실제로 이행됨.** ⚠️ 2부 무장 발화는 이 약조를 깨는 사건으로 설계해야 함. | 13·14화 |

## 🔥 집단행동 · 사건의 형식 (블록 4 「통문」, 1915-04 신력 한성)
> 민심 파문의 전개 순서와 규모. 후속 회차가 규모를 부풀리거나 성격을 바꾸면 즉시 모순이 나는 항목들.

| 항목 | 확정 내용 | 근거 |
|---|---|---|
| **전이 순서** | 통문 회람 → 베낀 밀지 유통 → **질문 사설**(공론화 방아쇠) → **철시** → **도회** → **해산 유혈** → 순교자 전설. 각 단계가 앞 단계를 명분으로 삼음. | 11~14화 |
| **철시(撤市)** | 사설 **이틀 뒤** 개시. 지물전 두엇 → 점심에 포목전 → **저녁에 종로 큰길의 절반**이 덧문. "별말 없이 닫되 다들 안다"(구호·조직 없는 무언의 항의). 백금옥의 장사도 멈춰 원산 짐이 창고에 재워짐. | 12화 |
| **도회(都會)** | 철시로부터 **나흘 뒤 낮**, 종로. 모델 = **무술년(1898) 만민공동회**("늙은이들이 기억하는 방식"). 조직자 = 차도형(밤마다 순서를 짬), 얼굴 = 최우성(흰 두루마기 = 소란을 상소로 바꾸는 방패). **차도형이 세운 규모는 백, 실제로 모인 것은 천 이상** — 계획이 규모에 무너짐. 구성: 짐꾼·등짐장수·저자 아낙·철시 상인·남촌 유생 등. | 13·14화 |
| **함경 연선의 발** | 소문에 물질적 발을 달아 준 것 = **함경 철도 부설 현장의 논밭 헐값 수용·부역 동원**. "나라 팔린 것이 제 논두렁 일이 됨". 9화의 함경 재가가 "억지로 찍히셨다"로 재해석됨. ⚠️ 협약 조문(제7·11조)과 현장 집행이 따로 논다는 것이 여기서 확정. | 13화 |
| **해산과 유혈** | 경무청이 오전 내내 포위·관망(흰 두루마기 때문) → 상부 명으로 **해가 기울 무렵 해산 착수** → 밀치기 → **출처 불명의 돌 하나**가 순검 이마에 피를 냄("젊은 축인지, 겁먹은 백성인지, 난이 되기를 바란 다른 손인지 아무도 모름") → 곤봉이 밀던 손에서 치는 손으로. 최우성은 "앉아서 버티라"(무술년 방식)를 외쳤으나 유혈 전에 **떠밀려 나감**. | 14화 |
| **사상자 규모** | **관 발표 셋 / 도성의 입 열 / 정확한 수는 아무도 모름.** ⚠️ **학살이 아니라 해산 충돌** — 후속 회차에서 대량 살상으로 부풀리면 톤 상한선 위반. **무기는 한 번도 사용되지 않음**(헛간에 그대로). | 14화 |
| **순교자** | 넘어진 노인을 일으키려다 죽은 **열예닐곱 살 등짐장수**. **이름이 끝내 알려지지 않음** — "이름이 없어서 더 완전한 순교자"(도성 전체가 저마다의 아들을 그 자리에 세움). 전설이 즉시 증식(열둘이라더라·노모 봉양 효자라더라·죽으며 상감마마를 불렀다더라). ⚠️ **이름을 나중에 부여하지 말 것** — 익명성이 이 장치의 엔진. | 14화 |
| **소문의 봉인** | 유혈 이후 도성은 묻기를 그만두고 믿기 시작함 — "소문은 반박으로 죽지 않았다. 소문은 피로 완성되었다." **이후 어떤 반증도 이 소문을 죽일 수 없음**(블록 5 이후 설계의 대전제). | 14화 |
| **유혈 전문** | 한성 러 공사관이 그날 밤 소요·사상자 보고 전문 작성 → 이튿날 아침 부산 경유 시베리아 전신 → **하루 만에 페테르부르크 도착**(평시 열이틀 대비. "나쁜 소식은 언제나 빨랐다"). | 14화 |
