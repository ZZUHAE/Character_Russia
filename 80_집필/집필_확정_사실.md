# 📒 집필 확정 사실 원장

> **소설이 창조·확정한 세계 사실의 등록부.** canon(00~60, 동결)도 아니고 진행 상태([[현재_정세]])도 아닌 것 — 원고가 *만들어낸* 조문·문구·수치·소품이 여기 산다. 30화에서 "그때 협약 몇 조였더라"로 원고를 다시 뒤지지 않게, 어기면 바로 모순 나는 것들을 모은다.
> **원칙: 원고=진실의 원천, 이 원장=색인.** 어긋나면 원고 승. append 위주(수정 시 사유 한 줄). 갱신 = 확정 파이프라인 ④.5([[_집필_규칙]] §7).

## 🔥 집단행동 · 사건의 형식 (블록 4 「통문」 1915-04 → 블록 6 「그 겨울」 1915-12~1916-04)
> 민심 파문의 전개 순서와 규모. 후속 회차가 규모를 부풀리거나 성격을 바꾸면 즉시 모순이 나는 항목들.
> ⚠️ 블록 6에서 이 파문은 **폭발에서 만성화로** 이행했다 — 표 뒷부분(옥사자~은화)이 그 국면.

| 항목 | 확정 내용 | 근거 |
|---|---|---|
| **전이 순서** | 통문 회람 → 베낀 밀지 유통 → **질문 사설**(공론화 방아쇠) → **철시** → **도회** → **해산 유혈** → 순교자 전설. 각 단계가 앞 단계를 명분으로 삼음. | 11~14화 |
| **철시(撤市)** | 사설 **이틀 뒤** 개시. 지물전 두엇 → 점심에 포목전 → **저녁에 종로 큰길의 절반**이 덧문. "별말 없이 닫되 다들 안다"(구호·조직 없는 무언의 항의). 백금옥의 장사도 멈춰 원산 짐이 창고에 재워짐. | 12화 |
| **도회(都會)** | 철시로부터 **나흘 뒤 낮**, 종로. 모델 = **무술년(1898) 만민공동회**("늙은이들이 기억하는 방식"). 조직자 = 차도형(밤마다 순서를 짬), 얼굴 = 최우성(흰 두루마기 = 소란을 상소로 바꾸는 방패). **차도형이 세운 규모는 백, 실제로 모인 것은 천 이상** — 계획이 규모에 무너짐. 구성: 짐꾼·등짐장수·저자 아낙·철시 상인·남촌 유생 등. | 13·14화 |
| **함경 연선의 발** | 소문에 물질적 발을 달아 준 것 = **함경 철도 부설 현장의 논밭 헐값 수용·부역 동원**. "나라 팔린 것이 제 논두렁 일이 됨". 9화의 함경 재가가 "억지로 찍히셨다"로 재해석됨. ⚠️ 협약 조문(제7·11조)과 현장 집행이 따로 논다는 것이 여기서 확정. | 13화 |
| **해산과 유혈** | 경무청이 오전 내내 포위·관망(흰 두루마기 때문) → 상부 명으로 **해가 기울 무렵 해산 착수** → 밀치기 → **출처 불명의 돌 하나**가 순검 이마에 피를 냄("젊은 축인지, 겁먹은 백성인지, 난이 되기를 바란 다른 손인지 아무도 모름") → 곤봉이 밀던 손에서 치는 손으로. 최우성은 "앉아서 버티라"(무술년 방식)를 외쳤으나 유혈 전에 **떠밀려 나감**. | 14화 |
| **사상자 규모** | **관 발표 셋 / 도성의 입 열 / 정확한 수는 아무도 모름.** ⚠️ **학살이 아니라 해산 충돌** — 후속 회차에서 대량 살상으로 부풀리면 톤 상한선 위반. **무기는 한 번도 사용되지 않음**(헛간에 그대로). | 14화 |
| **순교자** | 넘어진 노인을 일으키려다 죽은 **열예닐곱 살 등짐장수**. **이름이 끝내 알려지지 않음** — "이름이 없어서 더 완전한 순교자"(도성 전체가 저마다의 아들을 그 자리에 세움). 전설이 즉시 증식(열둘이라더라·노모 봉양 효자라더라·죽으며 상감마마를 불렀다더라). ⚠️ **이름을 나중에 부여하지 말 것** — 익명성이 이 장치의 엔진. | 14화 |
| **소문의 봉인** | 유혈 이후 도성은 묻기를 그만두고 믿기 시작함 — "소문은 반박으로 죽지 않았다. 소문은 피로 완성되었다." **이후 어떤 반증도 이 소문을 죽일 수 없음**(블록 5 이후 설계의 대전제). | 14화 |
| **★ 옥사자 다섯 (명부)** | 1915년 12월 3~11일 한성감옥서 사망. **김학수 62 부보상 / 박선일 44 지물전 사환 / 정순보 31 무직 / 최덕구 57 짚신 장수 / 이만춘 29 목수.** 수감 사유란 전원 *"사월 종로 집회 관련"*, 사망 사유란 전원 ***"환후(患候)로 물고(物故)"***(병명 미기재 = 관례). 무연고 2인(관 둘·인부 넷·성 밖 지게 삯 = 매장비 청구). 사인 = **급여미 한 홉·땔감 감축**(을묘년 봄 세출 조정, 도장 셋 중 하나가 서경환). 이만춘의 처 = 26세, 아이 둘. ⚠️ **이름이 있다는 것이 이 다섯의 기능** — 봄의 무명 순교자와 대칭(익명은 도성 전체의 아들이 되고, 유명은 옮겨 적을 수 있다) | [[020화_부고\|20화]] |
| **부고(訃告) 게재** | 황성신문 **광고·부고란**(제일 값이 싸고 제일 트집 안 잡히는 자리). 다섯의 이름·나이·직업·물고 날짜 + *"모두 한성감옥서. 사월 종로 집회 관련으로 수감 중이었음."* **논평·애도·개탄 0줄.** 모든 어휘가 감옥서 문구 그대로 — *"내가 붙인 것은 부고라는 두 글자뿐이오."* ★ **이 부고가 연선 집회의 낭독 대본이 됨**(22화). 발행 부수는 평소만큼이나 **"요새는 한 장을 열 사람이 봅니다"** | 20·22화 |
| **소요의 만성화 (형식)** | 봄의 폭발이 겨울의 지속으로 전환. ①**부분 철시**(1월 사흘·2월 나흘 — 가게별·시간대별로 이가 빠지듯. "통째로 닫으면 셈이 되는데 하루씩 닫으면 셈이 안 된다" = 상인에게는 사월보다 나쁨) ②**연선 역 앞 집회**(2월: 20 → 100 → **300명**. **무기·구호·투석 없음** — 서서 옥사자 다섯의 이름·나이·직업·사망일을 읽고 절하고 흩어짐) ③**밤의 사보타주**(침목 절취·교량 나사 6개·부목 1. **사람 대상 폭력 0**) ④소상(小祥) 국면의 **만장 마흔 몇**(4월, 이름 없는 순교자의 상주는 도성). ⚠️ 무장 발화는 여전히 없음 — 헛간 물건 봉인 유효 | 20·22·23화 |
| **연선 수비대의 산술** | 병력 **112명**으로 **역 11 + 교량 14 = 25개소** 방어 → 한 군데 넷 남짓, 밤낮 교대 시 **동시 배치 2명.** 지난주 어느 역 앞에 모인 인원 300. 다리와 역 사이 **2베르스타**에 마을. 연선 세 고을의 조선 순검 **41명**(일부 도성 차출, **"그자들도 그 마을 사람"**). 중대장 진술: ***"병력이 모자라면 손이 거칠어집니다. 많으면 서 있기만 해도 되는데, 모자라면 겁을 줘야 하거든요."*** | [[022화_넉_자\|22화]] |
| **영국 총영사관 조회장** | 프랑스어 3줄 — 본국 정례 보고 준비 중이며 *"함경 지방의 철도 연선에서 러시아 수비 병력의 활동 범위에 관하여 **몇 가지 문의가 접수**되었으므로 사실 관계의 확인을 청한다"*. ⚠️ **발신자 미기재가 요점**(관측의 압력, 개입 아님 = 국제 관중석) | 22화 |
| **★ 부의(賻儀) 속 일본 은화** | 자주 급진이 옥사자 다섯 집에 모두 부의를 돌림("넉넉히들 하십디다"). 이만춘의 처가 곡식으로 바꾸러 온 보자기에서 **일본 은화 여섯 닢**(앞면 용 문양). ★**단서는 '일본 돈'이 아님** — 개항 40년의 도성에서 화폐는 원래 섞여 다닌다. 단서는 ①**여섯 닢이 전부 같은 나라 것** ②**닳은 정도가 비슷함** ③**손을 적게 탄 새 돈**(= 한 손에서 한꺼번에 나와 몇 손 안 거침). 사랑방 공개 발화(곡물전 주인): ***"모인 돈이 왜 다 한 나라 것이오."*** → 아무도 대답 못 함 = **일본 개입 노출 1단계 점화**(정황·소문까지만. 다나카·조직·위조는 노출 0) | 22화 |
| **유혈 전문** | 한성 러 공사관이 그날 밤 소요·사상자 보고 전문 작성 → 이튿날 아침 부산 경유 시베리아 전신 → **하루 만에 페테르부르크 도착**(평시 열이틀 대비. "나쁜 소식은 언제나 빨랐다"). | 14화 |
