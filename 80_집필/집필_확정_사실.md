# 📒 집필 확정 사실 원장

> **소설이 창조·확정한 세계 사실의 등록부.** canon(00~60, 동결)도 아니고 진행 상태([[현재_정세]])도 아닌 것 — 원고가 *만들어낸* 조문·문구·수치·소품이 여기 산다.
> **원칙: 원고=진실의 원천, 이 원장=색인.** 어긋나면 원고 승. append 위주. 갱신 = 확정 파이프라인([[_집필_규칙]] §7).
> ⚠️ **2026-07-22 v3 재설계로 초기화.** v2 원고가 창조한 조문·수치·소품은 [[_OLD_v2]] 스냅샷에 있고 **효력이 없다.** 아래 부칭 등록부만 계승한다.

## 🇷🇺 러시아 인물 이름+부칭 등록부 (★ 2026-07-22 신설, [[_문체_바이블]] §14)
> 러시아 장면에서 **이름+부칭은 존중의 기본 형식**이다. 누가 부칭으로 부르고 누가 성으로만 부르는지가 관계다. 아버지 이름은 원고가 창조한 사실 — **한번 정하면 못 바꾼다.**

| 인물 | 이름+부칭 | 비고 |
|---|---|---|
| 니콜라이 2세 | **니콜라이 알렉산드로비치** | 실존 확정([[006화_깃발_아래\|6화]] 기사용) |
| 알렉산드라 황후 | **알렉산드라 표도로브나** | 실존 확정 |
| 마리아 돌마토바 | **마리아 아르카디예브나** | 아버지 아르카디 = 가문 당주. 그녀를 부칭으로 부르는 사람은 드물다(정적은 "부총리", 궁정은 "그 여자") — **부칭으로 부르는 자가 곧 그녀 편**이라는 지표로 쓸 것 |
| 세르게이 돌마토프 | **세르게이 보리소비치** | 아버지 보리스 = 마리아의 아우. 마리아는 그를 이름·약칭(세료자)으로 부른다 |
| 표트르 솔로비요프 | **표트르 이바노비치** | 총리. 각의에서 그를 부칭으로 부르는 것이 예우의 표준 |
| 블라디미르 가가린 | **블라디미르 미하일로비치** | 공작. 아랫사람은 "각하", 대등한 자만 부칭 |
| 알렉세이 라돈스키 | **알렉세이 페트로비치** | 노장. 병사들은 성으로만, 차르는 부칭으로 |
| 드미트리 멜니코프 | **드미트리 그리고리예비치** | 하급 봉직귀족 출신 — 부칭으로 불리는 것이 그에게는 획득한 자리다 |
| 예카테리나 라리오노바 | **예카테리나 니콜라예브나** | 살롱에서 부칭 호명이 곧 명부다 |
| 이반 체르니쇼프 | **이반 표도로비치** | 재무 |
| 드미트리 쿠즈민 | **드미트리 안드레예비치** | 두마 |
| 니콜라이 베르호프 | **니콜라이 세르게예비치** | 전 주한 공사 |
| 콘스탄틴 볼린스키 | **콘스탄틴 알렉산드로비치** | 확정([[019화_얼기_전에\|19화]]) |
| 표도르 젤레닌 | **표도르 일리치** | 경무 고문. 넉 달간 이 나라에서 아무도 그를 부칭으로 부르지 않았다 |
| 안나 마트베예브나 | (부칭이 이름) | [[028화_물을_자리\|28화]] |
| 파벨 세묘노비치 미로노프 | 확정 | [[020화_부고\|20화]] |

**★ 알렉산드르 박 — 부칭이 없는 사람**: 관보·러시아 문서상 「알렉산드르 박」. **부칭이 없다** — 아버지가 조선인이라 러시아식 부칭으로 옮길 이름이 아니기 때문. 러시아인들은 그를 부칭으로 부르지 못하고 "박 씨"·"통변관"·직함으로 부른다. 제국이 그에게 준 것은 이름 하나이고 **자리(부칭)는 주지 않았다.** ⚠️이것은 그의 정체성 균열의 물적 형식이며, [[_시험작_037화_레지스터\|젤레닌이 이덕수의 아버지 이름을 모르는 것]]과 같은 구조다.
