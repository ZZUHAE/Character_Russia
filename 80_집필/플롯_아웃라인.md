# 🗺️ 플롯 아웃라인 — 「북극성 아래」

> 소설에서 **무엇이 일어나는지**의 설계도. → [[_집필_규칙]] §5
> *(2026-07-11 플롯 인터뷰 + 딥 인터뷰(.omc/specs/deep-interview-novel-preflight.md)로 확정. 아래는 기준 항로 — 바꾸면 여기부터 고친다.)*
> **제목 확정: 「북극성 아래」** (2026-07-11)
> **★ 2026-07-13 재설계(사용자 방향)**: ①1부 엔딩 봉합→**파열**(베르호프 소환·볼린스키 부임을 1부 엔딩으로) ②'절반의 옥좌'(전시 조건부 입헌 쟁취) **기각** — 개연성 부족([[_고증_바이블]] ⓔ: 전시의 차르는 회의실에서 물러나지 않는다), 2부는 '요구와 거부'로 ③**한성발 대폭발이 입헌 전환의 방아쇠**(조선 사건과 메인 이벤트의 결합) ④점화 축 = 밀지 소문에 의한 **민심 이동**(차도형 테러 아님 — 상세 설계는 [[1부_플롯]]).
> **★ 2026-07-14 개정(리서치 기반, 사용자 방향)**: ①권력 구조 = **A안 — 차르 폐위 없음.** 전쟁 중엔 **통수권 박탈 요구·거부**까지, **전후(戰後) 권한 대폭 축소**로 실권 없는 상징 군주(군림하되 통치않음)를 완성. (승전은 군주를 지킨다 → 몰락 아닌 '전후 정산' — [[_고증_바이블]] ⓖ) ②러시아 엔진 = 선동·밀지 아니라 **차르 통수권 친정의 실제 실책**(전선 + 동아시아 이중 실패가 그의 개인 계정). ③2부 클라이맥스 조선 = 밀지 소문 → **자주 급진의 실제 무장봉기**(러 공격·민간 피해 → 일본 재개입 노출)로 격상 — 밀지는 엔진에서 도화선 첫 마디로 강등. ④**총리 [[표트르_솔로비요프]]를 내각파 대표 얼굴로 전면화**(마리아=자본·동력 뒤·옆, 솔로=총리 얼굴 앞).
> **★ 2026-07-14 2차 개정(사용자 지적: "전시에 실권을 안 잃으면 전후에 못 뺏는다")**: **전시 대위임 구조** 채택 — 실권 상실은 **2부 엔딩**(동방의 불+전선 동시 위기의 정점, '전쟁이 끝날 때까지'라는 체면 공식)에서 일어나고, **3부 = 궁정의 환수(還收) 시도를 저지 + 임시를 영구로 명문화**하는 환수 전쟁. 준거: [[_고증_바이블]] ⓖ(빌헬름 2세 그림자 카이저 1916~18 · 1906 환수 플레이북 · 처칠 1945).

## 전체 아크 ★

**"전쟁이라는 압력솥이, 혁명 대신 입헌을 낳는다 — 그리고 이번에는, 끝까지 간다."**

- **최종 상태 = 완전한 입헌 전환, 영국식.** 차르(니콜라이 2세)는 **퇴위 없이 존속**하되 "군림하되 통치하지 않는다"(A안, [[_고증_바이블]] ⓖ). 폐위·공화국이 아니라 **왕관을 살리고 실권만 비운다** — 승전국의 군주는 몰락하지 않으므로. 실권은 **2부 엔딩의 전시 대위임**으로 이미 넘어가고, 3부에서 궁정의 **환수 시도가 저지되며 임시가 영구로 명문화**되어 종결. 전제정 실질 종식 = 최종 목표 **완전 달성**. 내각 시점의 웹소설 문법: 승리는 진짜 승리다.
- **과정 = 요구와 거부의 축적 → 위기 정점의 대위임 → 환수 전쟁.** 전시의 요구는 **통수권 박탈**(아마추어 친정을 전문가 지휘로 되돌리라 — 친승리 논리라 '정권 교체'가 아니다)로 구체화되고, 차르는 이를 **거부**한다(준거: 1915 진보블록 '신임 내각' 요구 → 거부·정회·숙청, [[_고증_바이블]] ⓔ). 전시의 차르는 회의실에서 물러나지 않는다 — 그를 물러나게 하는 것은 회의가 아니라 **위기 정점**(동방의 불+전선 동시 위기 = 친정하는 두 전장이 동시에 불타는 지휘 마비)이며, 그때 **'전쟁이 끝날 때까지'라는 체면 공식의 대위임**이 강제된다(빌헬름 2세 1916~18 실증, ⓖ). 전후 정산 = 그 위임을 되찾으려는 **환수 전쟁**이고, 내각은 새로 뺏는 게 아니라 이미 쥔 것을 지킨다. *(구 '절반의 옥좌'=전시 조건부 입헌 쟁취 코드는 2026-07-13 기각.)*
- **★ 메인 이벤트 = 한성발 대폭발 (2부 클라이맥스).** 황실이 (내각 반대를 무릅쓰고) 고른 강압 노선(볼린스키)이 온건 자주파를 급진으로 몰아 **실제 무장봉기**를 낳는다 — 자주 급진의 무력이 러시아를 치고, 진압(=차르 통수권)이 민간 피해를 키우며, 배후의 **일본 재개입이 노출**된다. ⚠️**비난 벡터의 핵심**: 적은 반란군+일본이되 **범인은 궁정의 선택**(강압 노선을 고른 것 + 통수권의 진압 실책)이어야 한다 — 안 그러면 외적 배후 반란이 오히려 차르 중심 결집을 부른다. 봉기는 **군사적으로 진압(독립 없음)되되 정치적으로 차르를 폭파**한다. 전선 위기·전쟁 피로와 공명하며 황제 책임론의 결정타 — **조선의 사건과 입헌 전환이 한 줄의 인과로 엮이는 지점.**
- **전쟁 = 원역사 유사 소모전.** 혁명은 터지지 않고 참전은 지속된다. 1905를 흡수한 개혁이 압력을 '입헌 요구'로 수렴시킨다. (승전 작업 가정: 1918년경 독일 소진 — 방향 안 가르는 디테일, 필요시 조정)
- **인과의 척추**: 소모전=물량·조직전 → 전쟁을 먹이는 건 내각(군수·재정·철도) → 길수록 "내각 없인 하루도 못 싸우는 나라" → 차르 친정은 책임만 축적 → **입헌은 쿠데타가 아니라 청구서로 온다.** 책임론의 3계정: ①**전선**(친정 = 패배·물자난이 황제 개인 계정) ②**동방**(강압 노선 = 황실이 직접 고른 프로젝트의 파산) ③**내정**(요구 거부·인사전 = 통치 불능의 광고). 셋이 3부에서 한 책상에 모인다.
- **내각파의 얼굴 구조 (2026-07-14)**: [[마리아_돌마토바]]는 자본·군수·민심 지렛대를 쥔 **실세 동력**이나, 여성·신흥 산업 외부자·궁정 신임 부재라 스스로 전면 수반이 못 된다. 그래서 **총리 [[표트르_솔로비요프]]가 내각파의 대표 얼굴로 전면에 선다** — 능력과 정치적 상징성은 갖췄으나 자본도 민심도 제 기반이 아니어서(그 무게는 경륜·궁정 신임뿐) 마리아와 **구조적 동맹**. 마리아 = 뒤·옆의 동력, 솔로비요프 = 앞의 총리 얼굴(두마·궁정·공개 석상에서 내각을 대표해 차르·황실파와 맞선다). 이 '얼굴'이 전면에 설수록 그가 쓰러질 때의 값도 커진다(**인적 값 후보 강화**).
- **성공의 값 = 과정의 비용** (뒷맛의 냉소 ❌ — 변질 코드는 기각됨):
  - ⓐ **암흑 구간** (2부): 황실 반격으로 개혁이 크게 밀리는 밑바닥 — 반등의 발판.
  - ⓑ **인적 값 1명**: 내각 진영 기둥 중 하나가 입헌의 문턱에서 쓰러진다. 누구인지는 **2부 축적으로 결정** (후보군: [[표트르_솔로비요프]] / [[드미트리_멜니코프]] / [[알렉산드르_박]] / [[예카테리나_라리오노바]]).
- **톤 배분**: 교착의 무게([[톤_바이블]])는 1~2부 **과정**의 문법. 3부는 그것을 진짜로 뚫는다 — 단 합법 절차로(쿠데타·혁명 없는 쟁취, [[알렉세이_라돈스키]]의 합법충성=군 중립이 전제조건).
- **대한제국 라인 = 회색 줄타기 → 열린 문.** 끝까지 어느 편에도 완전히 서지 못하는 공존이되, 입헌 러시아의 탄생이 협력 노선을 공고화하며 숨통이 트이는 **희망 코드**로 닫는다 — 깔끔한 해방이 아니라, 회색 줄타기가 '열린 문' 앞에서 끝나는 여운.

## 부(部) 구상 — 3부 골격

### 1부 — 압력솥 (1915)
- 세 무대([[러시아_내정]]·[[동아시아]]·[[동부전선]])와 군상 소개. 소모전의 질감, 교착의 정치 지형, "포탄이 곧 정치" 등식.
- **1부 클라이맥스 = 밀지 파문 → 민심 이동 → 파열 ★** (2026-07-13 재설계): [[고종]]의 밀지 소문이 충군(忠君)의 방향을 뒤집어 반러 민심이 실체를 얻는다(통문·철시·도회 — 설계는 [[1부_플롯]] '민심 파문 설계'). [[일본의_조선_공작망]]은 증폭기(독자만 앎). 본토 인사전에서 내각 첫 패배 — **[[니콜라이_베르호프]] 소환, [[콘스탄틴_볼린스키]] 부임 발령**으로 1부를 닫는다. 동아시아發 사건이 본토 저울을 실제로 기울이는 첫 시범.
- 1부 엔딩: **강압의 시대 개막(다크)** + 독일 대공세 전조(2부 직결). 값과 불씨 — 협력 노선 밑바닥 / [[서경환]]의 문서·[[백금옥]]의 침묵 카드·[[알렉산드르_박]]의 결심 = 3부 복원의 씨앗.
- 상세 블록 설계: [[1부_플롯]]

### 2부 — 저울이 기운다 (1916~17경)
- **강압의 시대**: 볼린스키의 한성(검열·검거·이권 강행)이 민심을 지하로·급진으로 몰아넣고, 본토는 독일 대공세 방어전으로 "내각 없이는 전쟁 불가"가 공고화.
- **요구와 거부**: 입헌 요구의 공식화(두마·여론·상공업의 합창) → **차르의 거부·정회·숙청** — 이 거부가 **암흑 구간 ★**(황실 반격 최고조: 오흐라나·인사전·정교회 동원, 개혁의 밑바닥)을 연다. 인적 값의 복선이 여기서 익는다.
- **★ 2부 클라이맥스 = 한성발 대폭발**: 강압에 몰린 자주 급진의 **실제 무장 발화**(2026-07-14 격상) + 도회·철시·지방 확산 + 진압(=통수권)의 유혈·실패 + 공작망 개입 노출 + 국제 스캔들 → **"황제가 고른 노선이 제국의 동방을 불태웠다"** — 황제 책임론의 결정타.
- **★ 2부 엔딩 = 전시 대위임 (2026-07-14)**: 동방의 불과 전선의 동시 위기가 지휘 마비를 낳고, 군·내각·두마가 한목소리로 위임을 강제 — 통수권은 스타프카로, 국정은 책임내각으로, 차르는 재가만("전쟁이 끝날 때까지"). 가가린은 1906 플레이북("지금 내주고 승전 뒤 거둔다")으로 수용을 진언 — **그 미소가 3부의 훅.** 저울이 '기운다'가 여기서 문자 그대로 넘어간다.

### 3부 — 정산 (승전 전야~직후) = 환수(還收) 전쟁 ★ (2026-07-14 재정의)
- **승전이 온다 — 내각의 위임 체제 아래서.** 월계관이 누구 것인지 온 나라가 본다. 그리고 궁정이 1906 플레이북대로 움직인다: *"비상은 끝났다. 전시 위임을 왕관으로 환수한다."* — **3부의 최종 대결 = 환수 시도 vs 저지.** 정산은 수동적 회계가 아니라 능동적 방어전이고, **"이 전쟁은, 이 제국은 누구의 것인가"**(전공 귀속 다툼, [[러일전쟁_러시아_승리]]의 대전판)가 그 전장이다. 공개 탄핵 국면(밀류코프식 연설의 본작판 — [[_고증_바이블]] ⓔ)은 저지전의 정점 무기.
- **환수가 실패하는 이유 5 (= 1~2부에 심는 씨앗)**: ①월계관이 가시적으로 내각 것(승전 자체가 위임 체제의 실적) ②동원 해제 — 귀향 병사들이 원하는 건 재건·약속 이행이지 왕정 복고가 아님 ③**라돈스키의 합법주의가 이번엔 차르를 벤다** — 법이 위임을 말하므로, 궁정의 비상수단 요청을 노장이 "법의 절차를 밟으시라"로 거절(합법충성의 대형 payoff) ④협상국·채권단은 이미 내각 정부와 거래(전후 청산·차관·해협) ⑤**동방 계정** — 조선 사태+일본 공작 노출이 궁정의 판단력을 신용 파산시킴. **[[서경환]]의 문서가 여기서 물증으로 채택 = 조선 라인의 능동 매듭**("사람은 갈려도 문서는 남는다"의 대형 payoff — 조선 인물의 종이가 러시아 입헌을 미는 손이 된다).
- **종결 = 차르의 마지막 능동적 선택.** 이길 수 없는 환수전을 *싸우지 않기로 결정*하고 명문화에 서명 — "군림하되 통치하지 않는다." **인적 값 1명의 희생이 그 서명 직전에 놓인다**(죽음이 서명을 산다). 평생 결정을 미뤄온 남자의 유일한 능동이 '권력을 쥐지 않기로 결정하는 것'이라는 아이러니 — 완승 사이다가 아니라 값을 치른 승리.
- 한성: 볼린스키 소환·협력 노선 복원(박·베르호프 라인의 정당성 입증) = 입헌 러시아의 탄생과 맞물리는 **열린 문** — 그 앞의 회색 줄타기로 닫음.

## 1화 진입점 ★
- **작중 시점: 1915년 2월(구력), 늦겨울.**
- 구성(장면 수는 1만자 분량에 맞게 유연 — 2026-07-11 분량 상향 반영): **동부전선 보급 종착역/참호에서 연다** — 포탄 기근의 냄새, 전쟁의 얼굴(이름 있는 단역 초점). → 페테르부르크로 카메라 이동, **군수 보고를 받는 [[마리아_돌마토바]]** 초점으로 닫는다. "포탄이 곧 정치" 등식을 1화에 박는 설계.
- 1화부터 "동쪽의 작은 불씨" 코드(한성·밀지의 예감)를 한 줄 심는다. 한성 본격 개방은 3화경.

## 연재 호흡
- 부당 15~25화 감각(유연). 스레드는 긴장 높은 쪽에 카메라 — 단 한 스레드가 6화 이상 방치되면 점검.

## 플롯 3층 구조 (운용)
- **큰그림**(이 문서) → **부 블록**(`80_집필/부_플롯/` — 부마다 에피소드 블록 3~6개, 블록=3~5화) → **회차**. 각 층 모두 네모 초안 → 너 승인. ([[_집필_규칙]] §5)
