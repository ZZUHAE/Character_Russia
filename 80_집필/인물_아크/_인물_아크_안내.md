# 👥 인물 아크 — 사용 규칙 v2 (살아있는 원장)

> **canon(10_인물) = 인물의 설계도(1915 동결), 아크(여기) = 소설 속에서 그 인물이 걸어간 길(갱신).**
> 이 분리가 canon↔state 원칙의 인물판이다 — 소설을 아무리 진행해도 동결 배경은 깨끗하게 남는다.
> **★ 드리프트 해독제: 원고(90_원고)=진실의 원천, 아크=색인. 어긋나면 원고가 이긴다(아크를 고친다).**

## 규칙
- 파일명 = canon 인물 노트 동명 뒤 `_아크`. 예: `마리아_돌마토바_아크.md`.
- 템플릿: `_템플릿/인물_아크.md` (v2, 7섹션). frontmatter = `type: 인물아크` · `status: 가동/퇴장/사망` · `character`(canon 역링크) · `tier`(주역/조연) · `last_ep`(마지막 반영 확정회차).
- **Tier**: `주역` = 7섹션 전부 / `조연` = ①스냅샷·⑤로그만 필수(나머지 생기면 채움).
- 담는 것: ①현재 스냅샷 ②지식 원장(화약별 안다·의심·모른다) ③관계 변화분 ④목소리 앵커+후렴 이력 ⑤로그(append) ⑥예정 아크 ⑦❓.
- 담지 않는 것: canon에 이미 있는 설정(중복 금지 — 링크로 가리킨다). 관계 원장은 canon 관계망의 **변화분만**.
- **canon과 모순 금지.** 인물이 canon 성격·입장을 *벗어나게 변할* 만큼 큰 사건이면 국면 전환 — [[_집필_규칙]] §5 보고 대상.

## 갱신 회로 (핵심 — 이게 없으면 또 방치된다)
- **확정 파이프라인 6단계**([[_집필_규칙]] §7): 연대기 → 현재_정세 → 타임라인·스레드 → **④.5 인물 아크 + [[집필_확정_사실]]** → 모순 체크 → 커밋. **블록 확정당 1회**(회차마다 아님).
- ④.5 절차: 블록 내 초점 인물+크게 흔들린 비초점 → 각 아크 스냅샷 덮기·지식 원장 갱신(신규 화약은 아래 마스터에도 등록)·관계 변화분·후렴 이력·로그 append·`last_ep` 갱신 → 신규 초점 인물은 아크 생성 → lint.
- **stale 감시**: [[_집필_MOC]] 대시보드(뒤처진 아크 맨 위) + **lint 하드체크**(가동 아크 `last_ep`가 최신 확정회차−5 초과 지연 = 위반). 진짜 파수꾼은 lint.
- **검수 연동**([[_검수_체크리스트]] 축④⑤): 판정 근거 = 아크 지식 원장·목소리 앵커·[[집필_확정_사실]] 대조 + 스팟 원고 확인.
