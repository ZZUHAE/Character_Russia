# 🔢 교착 균형 정량 모델 (Balance Index)

> 이 세계의 엔진인 '전제 vs 개혁의 교착'([[톤_바이블]])을 하나의 숫자로 잡는 도구. 새 사건·충격이 "균형을 얼마나 흔드는가"를 한 잣대로 환산해, 작가가 긴장도를 일관되게 통제하게 한다. 제도적 본체는 [[차르_전제정과_입헌_긴장]].
> ⚠️ 이 수치는 '못박는 사실'이 아니라 **드라마 통제용 기준계**다. 톤·서술이 먼저, 숫자는 그 정합성 점검용.

## 공식
```
균형지수  B = 50 + k · (L_C − L_T)      (k = 40)
  L_C (내각/개혁 측 힘) = 0.45·전쟁수행 + 0.30·도시여론 + 0.25·재정금융
  L_T (황실/군주정 측 힘) = 0.40·군충성 + 0.35·전통정당성 + 0.25·법적최종권
  각 레버는 0~1로 정규화. B = 50 이 '완전 교착'.
```
- **교착 회랑(corridor): B 40~60.** 이 안에 있으면 "팽팽한 교착" 유지. 벗어나면 한쪽이 결정적 우위(드라마의 국면 전환).
- 가중치는 CANON의 무게감 반영: 내각의 명줄=군수·여론, 황실의 명줄=군·교회.

## 잠금장치 (톤 상한선)
- [[알렉세이_라돈스키]]의 **합법 충성**이 제약: 군충성 레버는 정치 압력으로만 작동하고 **쿠데타로 전환되지 않는다.** → B가 무력으로 한쪽에 폭주하는 것을 막는 상한선. 권력은 응접실에서 갈리지 전차로 갈리지 않는다.

## 전쟁의 진자 (★ 차르 친정)
- 차르가 최고사령관을 친임하면서([[1차_세계대전]]), **전황이 양쪽 레버를 동시에 흔드는 진자**가 된다. 전쟁은 더 이상 내각만의 'L_C 전쟁수행 레버'가 아니라, 황실의 'L_T 전통정당성'까지 함께 건드린다.
  - **승전 → 황실로**: 차르의 위신이 오른다(전통정당성↑). 전쟁수행 성과는 내각 몫이기도 하나, 통수의 영광은 친정한 차르에게 귀속된다.
  - **패색 → 내각으로(완충적)**: 전쟁수행 레버↓로 내각 명줄이 깎이지만, **친정한 차르가 패전 책임을 personally 떠안아 전통정당성도 함께↓** — 그래서 패색의 순(純) 스윙은 한쪽으로 깨끗이 가지 않고 양면적이다. "차르가 지휘하고 내각이 먹이는" 구조라, 전쟁의 실패는 누구도 온전히 면책되지 않는다.
- 함의: 전황은 1915 현재 '안정된 교착'이라 진자가 중앙에 멎어 있으나, **B를 회랑 밖으로 밀어낼 가장 큰 단일 변수**다. 큰 전과·대패가 날 때마다 이 진자를 먼저 따져 B를 재계산한다.

## 1915 기준값
| 측 | 레버 | 값 | |
|---|---|---|---|
| 내각 | 전쟁수행 | 0.65 | 강한 군수·후방 |
| 내각 | 도시여론 | 0.62 | 개혁 명망(우세) |
| 내각 | 재정금융 | 0.55 | 외채 의존이 발목 |
| 황실 | 군충성 | 0.60 | 장교단 충성(라돈스키) |
| 황실 | 전통정당성 | 0.55 | 교회·농촌 |
| 황실 | 법적최종권 | 0.48 | 칙령·거부권·오흐라나 |

→ L_C ≈ 0.62, L_T ≈ 0.55, **B ≈ 52.8** (내각 약간 우세 — CANON "여론은 내각에 다소 기움"과 합치, 회랑 한가운데).

## 충격 시나리오 (민감도)
| 시나리오 | 흔들리는 레버 | B | 해석 |
|---|---|---|---|
| 기준(1915) | — | **52.8** | 팽팽한 교착 |
| 대독 전선 붕괴 | 전쟁수행 0.65→0.40, (차르 친정) 전통정당성 0.55→0.50 | **≈49** | 내각 명줄 깎이나 차르도 위신 타격 → 스윙 완충, 회랑 내 |
| 큰 승전(남부·캅카스) | (차르 친정) 전통정당성 0.55→0.63 | **≈54** | 황실 위신↑로 기우나 회랑 내 |
| 전선붕괴 + 흉작 | +전통정당성 0.55→0.65, 재정 0.55→0.45 | **≈46** | 회랑 하단 접근(곡물 '중의존' 임계) — *겹쳐야 임계* |
| 마리아 실각 | 도시여론 0.62→0.40, 재정 −0.10 | **≈49** | 황실로 기우나 군 합법충성이 폭주 차단 |
| 금본위 이탈 강행 | 재정 급락·여론 −, 단 전비 숨통 | (미래 변수) | [[외채와_금본위_국가부채]]의 열린 뇌관 |
| 일본 재무장·극동 교란 | 외부 압력 → 전쟁수행 여력 잠식 | 소폭↓ | 후방 불안이 내각 명줄을 갉음 |

핵심: **단일 충격으로는 회랑을 잘 못 벗어난다. 둘 이상 겹쳐야 임계.** 이것이 "교착은 근대화로도 못 깬다"는 톤을 숫자로 지키는 장치.

## 사용법
새 사건/설정을 만들 때: ① 어느 레버를 ±얼마 흔드나 → ② B 변화 계산 → ③ 회랑(40~60)을 벗어나면 "국면 전환급 사건"이니 신중히. 배경 단계에선 B를 회랑 안에 유지.
