# 🔢 거시 회계 시트 (1915, 명목 추정)

> 제국 경제를 한 장에 정합하게 묶은 단위계. 노트마다 "구체 수치 ❓"로 남겨둔 곡물 비중·외채 부담·전비·인플레를 같은 기준점으로 환산한다. [[교착_균형_모델]]의 '재정·전쟁수행' 레버에 실측 근거를 댄다.
> ⚠️ 못박는 사실이 아닌 **드라마용 기준 추정치**. 실제사 1900년대 러시아 통계를 앵커로, 본작 분기(개혁 성공·승전·극동 안정)를 보정한 값.

## 실제사 앵커 (1913 러시아)
- NNP 약 200억 루블 / 국가예산 약 30~35억 루블 / 대외채무 약 80~90억 루블(세입의 2~3배) / 전체 수출 중 곡물 약 40%.

## 본작 1915 추정 시트
| 항목 | 값 | 비고(본작 보정) |
|---|---|---|
| NNP(국민순생산) | **~230억 R** | 개혁 성공으로 실제사보다 소폭 상향 |
| 국가세입 | **~38억 R** | |
| 전비(전시 지출) | **~22억 R (세입 58%)** | 1차대전 총력동원 |
| 군수 예산 비중 | **50%+** | [[전시경제_총력동원]] |
| 수출 구성 | **곡물 35% / 산업·군수재 30% / 기타 35%** | 곡물 '중의존'을 수치로 정의(40%→35%) |
| 외채 총액 | **~95억 R (GDP 41%, 세입 2.5배)** | 협상국(프·영) 의존, 승전으로 신용 약간 양호 |
| 연 외채 원리금 | **~6억 R (세입 16%)** | [[외채와_금본위_국가부채]] '황금 족쇄' |
| 금준비 커버리지 | **~100%선 유지** | 금본위 이탈 문턱 *미도달*(1915) |
| 누적 인플레 | **1914→1915 +20%** | 전시 물가고 → 민생 압박 |

## 핵심 비율이 말해주는 것
- **곡물 35%** = "유일축은 아닌 주요축" → 흉작 한 번에 즉각 붕괴는 아님(완충). [[농업경제와_곡물수출_구조]]의 '중의존' 정의값.
- **외채 GDP 41% / 세입 2.5배** = 감당 가능하나 채권국=동맹국이라 전선에서 못 빠지는 정치적 족쇄.
- **전비 58% + 인플레 20%** = 후방은 버티나 민생·중소상공·농촌이 쥐어짜이는 구조 → [[신흥_부르주아와_상공계층]] 중소층 부동화, 농촌 불만의 회계적 근거.

## 일본 배상금 (중간 강도, 수치화)
- 명목 **~9억 엔**, **1906~1915 10년 분할**(연 ~0.9억 + 표준 이자).
- 일본 한 해 세입(~6억 엔) 대비 **상회하나 파탄은 아닌 부담** → "설욕심 깊되 도발은 미뤄짐"의 회계적 근거. [[하얼빈_강화조약]]
- 최종 입금년 **1915 = 해군제한 만료년**과 겹침 → "빚 다 갚는 해 = 손 푸는 해"의 극적 정합. 상세 타임라인은 후속 (e) 캘린더 후보로.

## 파생 연결 (이 시트에서 뻗어나갈 것)
- (c) 군사력 매트릭스 — 군수 예산 50%·포탄 생산을 러:일:독 비교로.
- (e) 배상금/이권 캘린더 — 위 9억 엔 분할표를 연도별로.
- (f) 대한제국 경제 — 대한제국 GDP를 러 NNP의 3~4%(~7~9억 R)로 환산.
